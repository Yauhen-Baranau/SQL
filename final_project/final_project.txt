В качестве финального проекта за основу взята тема "Расписание школьных занятий".

Создана база данных school_db
База данных состоит из 8 таблиц с заполненными данным:

- таблица classes (список классов школы )
- таблица audience (список школьных кабинетов )
- таблица teachers (список учителей, работающих в школе )
- таблица activities (список школьных предметов )
- таблица students (список учащихся )
- таблица students_log (таблица, содержащая информацию о дате зачисления учащегося )
- таблица students_state (таблица, содержащая информацию какой ученик к какому классу относится )
- таблица schedule_activities (таблица, содержащая информацию о расписании занятий в школе, по какому предмету, в какой аудитории, какой преподаватель, у какого класса )

В проекте создано представление, триггер и индекс на таблицу students.
